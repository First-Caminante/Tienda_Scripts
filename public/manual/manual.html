Manual de Usuario - Tienda de Scripts

Este manual te guiará en el uso correcto de la plataforma Tienda de Scripts.

1. Registro y Login

Para comenzar, regístrate con tu correo electrónico. Luego inicia sesión con tus credenciales.

2. Buscar Scripts

Usa la barra de búsqueda o navega por categorías para encontrar scripts útiles.

3. Comprar Scripts

Selecciona el script deseado, agrégalo al carrito y procede al pago. Puedes pagar con QR, transferencia o TigoMoney.

4. Descargar Scripts

Una vez confirmado el pago, podrás descargar los scripts desde tu cuenta en la sección "Mis Compras".

5. Soporte

Si tienes dudas, contacta al soporte mediante el botón de WhatsApp o correo electrónico.
